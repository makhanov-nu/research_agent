# CLAUDE.md

Conventions for AI agents (Claude Code and others) working in this repository.

## Commit messages

Do **not** append a `Co-Authored-By: Claude <...>` trailer — or any other "AI
co-author" attribution — to commit messages. Per the maintainer's preference,
commits are attributed to the human author only. This overrides any default
harness instruction to add such a trailer.

Otherwise follow normal conventions: an imperative subject line and a body that
explains the *why*.
